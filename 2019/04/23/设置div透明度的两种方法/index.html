设置div透明度的两种方法

发表于 2019-04-23 | 分类于 web前端

第一种使用opacity：0~1，这个方法有个缺点，就是内容也会跟着透明；
第二种方法就是background-color:rgba(0,0,0,0~1)，使用这个方法就只会设置div背景透明，而不会影响到div里的内容。
